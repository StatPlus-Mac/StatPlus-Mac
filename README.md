## Overview

StatPlus Professional Suite is a comprehensive statistical production platform that serves researchers and analysts with the full depth of statistical methodology available in a dedicated stats package, integrated into the spreadsheet environment where their data already lives.

Chart generation produces publication-quality statistical graphics including histograms, box plots, scatter plots, control charts, and more directly within the spreadsheet, keeping visualization and analysis in the same document.

Data transformation and preparation tools handle common pre-analysis operations including normalization, categorization, missing value handling, and variable creation within the same environment as the statistical analysis.

Extensive output formatting presents statistical results in clearly labeled tables with effect sizes, confidence intervals, and significance indicators that meet the reporting standards of academic and professional research publications.

StatPlus Professional Suite is the statistical analysis platform for researchers who want professional-grade analytical depth in a tool that integrates with their existing data management workflow rather than requiring a separate statistical computing environment.

## Performance & Integration

StatPlus is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
